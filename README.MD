## Games

Braincup includes **36** activities: **32** scored mini-games plus **4** extras.

### Math
- **Mental Calculation** — Follow expressions; use each result as the base for the next
- **Chain Calculation** — Solve complex formulas step by step
- **Sherlock Calculation** — Reach the goal using only the given numbers and operators
- **Fraction Calculation** — Solve fractions
- **Value Comparison** — Pick the expression with the highest value
- **Bubble Sum** — Add numbers on floating bubbles (later rounds blink)
- **Quick Sum** — Numbers flash one at a time; enter the total

### Logic
- **Anomaly Puzzle** — Find the figure that stands out by color and shape
- **Path Finder** — Follow arrow instructions from a marked start
- **Colored Shapes** — Add points from correct statements
- **Mini Sudoku** — Small grid: each row, column and block once per digit
- **Mini Chess** — Finish a chess scenario as white against the CPU
- **Solo Chess** — Capture pieces until only the king remains
- **Lights Out** — Turn off every light; taps toggle neighbours
- **Sliding Puzzle** — Slide tiles into order
- **Shikaku** — Divide the grid into rectangles matching their numbers
- **Nurikabe** — Paint the sea around numbered islands (no 2×2 blocks)
- **Cat Queens** — One cat per row, column and color zone; no touching
- **Knot** — Connect colored dot pairs without crossing; cover every cell
- **Tower of Hanoi** — Move the stack to the right peg
- **Wordle** — Guess the five-letter word in six tries

### Memory
- **Visual Memory** — Memorize shapes, then identify the hidden one
- **Ghost Grid** — Watch tiles light up; tap them back in order
- **Orbit Tracker** — Track highlighted balls as they move
- **Digit Memory** — Memorize digits, solve a quick math problem, recall
- **Spot the New** — Tap the animal you have not seen before
- **N-Back** — Watch shapes flash; tap the one at the asked position

### Perception
- **Pattern Sequence** — Find the next figure in the pattern
- **Color Confusion** — Tap cells where the word matches its font color
- **Flash Crowd** — Tap the side that flashed more dots
- **Schulte Table** — Tap numbers in order as fast as you can
- **Flags** — Identify countries by their flags against the clock

### Extra modes
- **Matchstick Riddles** — Move matchsticks to fix equations
- **Peg Solitaire** — Jump pegs until one remains
- **Sudoku** — Classic full-size Sudoku with multiple difficulties
- **Chess** — Full chess vs CPU or pass-and-play

## Goal
Build an open source math, memory and focus improvement app. The app itself consists of a bunch of small games. Right now I try to keep the games as simple as possible to allow the implementation for the terminal client.
